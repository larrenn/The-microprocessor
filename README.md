## Компонентная база STM32

### Основные компоненты системы

**Микроконтроллер STM32F103C8T6:**
- **Ядро**: ARM Cortex-M3 72MHz
- **Память**: 64KB Flash, 20KB RAM
- **Периферия**: 37 GPIO, 10 АЦП, 2xSPI, 2xI2C, 3xUSART
- **Назначение**: Центральный процессор системы

**Система питания:**
- **Стабилизатор**: AMS1117-3.3V
- **Преобразование**: 5V → 3.3V
- **Максимальный ток**: 800mA
- **Защита**: Thermal protection

**Тактовая система:**
- **Основной кварц**: 8MHz HSE
- **RTC кварц**: 32.768kHz LSE
- **Нагрузочные емкости**: 22pF

## Среда разработки STM32

### Сравнение сред разработки

| Среда | Тип | Плюсы | Минусы |
|-------|-----|-------|--------|
| **STM32CubeIDE** | Официальная IDE | Полная интеграция с HAL | Требовательна к ресурсам |
| **PlatformIO** | Расширение VS Code | Поддержка 1000+ плат | Требует настройки |
| **Keil MDK** | Коммерческая | Высокая оптимизация | Дорогая лицензия |
| **Arduino IDE** | Упрощенная | Простая установка | Ограниченный функционал |
